# Event Platform

A modern and elegant website for managing and showcasing events, featuring a responsive design, intuitive navigation, and interactive elements. The platform allows advanced content management and provides a pleasant experience on any device.

## Main Features
- Responsive and modern design
- Intuitive navigation
- Interactive elements (e.g., newsletter signup, checkout, confirmation)
- Sections for categories, featured events, event details, etc.
- Easy content management

## Technologies Used
- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Docker](https://www.docker.com/) (optional)

## Local Installation
1. Clone this repository:
   ```bash
   git clone <your-repo-url>
   cd event_platform
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Access the app at `http://localhost:3000`

## Running with Docker (optional)
1. Make sure you have [Docker](https://www.docker.com/) installed.
2. Run:
   ```bash
   docker-compose up --build
   ```
3. Access the app at `http://localhost:3000`
